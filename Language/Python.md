### Print 함수
``` python
print("Hello World")
print("파이썬 시작!")

# 출력 
>> Hello World
>> 파이썬 시작!
```

### 연산자
``` python
tmp_1 = 2 + 3   # 덧셈 연산자
tmp_2 = 2 * 3   # 뺄셈 연산자
tmp_3 = 2 / 3   # 나누기 연산자
tmp_4 = 2 // 3  # 몫 연산자
tmp_5 = 2 % 3   # 나머지 연산자
tmp_6 = 2 ** 3  # 제곱 연산자

# 2와 3을 변수 처리해서 계산도 가능
n1, n2 = 2, 3
tmp_7 = n1 + n2

tmp_arr = [tmp_1, tmp_2, tmp_3, tmp_4, tmp_5, tmp_6, tmp_7]
for result in tmp_arr:
  print(result)

# 출력
>> 5
>> 6
>> 0.6666666666666666
>> 0
>> 2
>> 8
>> 5
```
