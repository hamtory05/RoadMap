## 재귀 (Recursion)
* 어떠한 것을 정의할 때 자기 자신을 참조하는 것
* 자신이 자신을 호출하는 것
* 재귀함수 사용시 반드시 탈출 조건이 있어야 stack overflow를 방지할 수 있다.

``` python
# 간단한 예시
def Recursion(n):
    if n == 0:
        return
    else:
        print(n)
        Recursion(n - 1)


Recursion(3)

# 출력 값
>> 3
>> 2
>> 1    
```

### 피보나치 수열
* 피보나치 수열을 재귀함수로 표현하면 굉장히 느림
* 하지만 재귀함수의 좋은 예시는 피보나치

``` python
def fib(n):
    if n < 2:
        return n
    else:
        return fib(n - 1) + fib(n - 2)


result = fib(10)

print(result)  # 55

```

``` python
# for문을 이용한 피보나치 빠른 계산
def fib(n):
    if n < 2:
        return n

    a, b = 0, 1
    
    for _ in range(2, n + 1):
        a, b = b, a + b

    return b
```
